Có 2 cách để chạy chương trình này trên máy tính :

a) CÁCH 1: Chạy chương trình bằng trình biên dịch (IDE) sử dụng ngôn ngữ Java
 - Máy tính cần cài đặt JavaSE 1.8 trở lên.Nếu máy tính chưa có java phù hợp, có thể thực hiện theo các bước như sau:
  + Bước 1:

      Tiến hành download JDK tại địa chỉ : http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
  + Bước 2:
      Mở file .exe vừa tải về và tiến hành cài đặt như một phần mềm thông thường.

 - Cài đặt một trình biên dịch ngôn ngữ Java (ví dụ: NetBeans, Eclipse, ... ). Ở đây, chúng tôi sử dụng IDE NetBeans (phiên bản mới nhất là 8.2).
   Nếu máy tính chưa được cài đặt NetBeans ,có thể làm theo các bước như sau:
  + Bước 1:
      Tiến hành download NetBeans tại địa chỉ : https://netbeans.org/downloads/ (Hãy chọn phiên bản phù hợp nhất với máy tính của bạn)
  + Bước 2:
      Mở file .exe vừa tải về và tiến hành cài đặt.
( Lưu ý: Bạn nhất thiết phải cài đặt JDK trước khi cài đặt bất cứ 1 trình biên dịch nào của Java mà không được làm ngược lại. Nếu không sẽ có lỗi xảy ra và bạn sẽ không thể sử dụng chương trình của chúng tôi)
 
- Sau cùng, khi có đủ các điều kiện ở trên, có thể thử một trong 2 cách dưới đây để chạy Project trên NetBeans IDE.

   + Cách  1:

       
- Để có thể chạy Project trên NetBeans IDE, cần import Project (ở cùng thư mục với file readme này) vào trong NetBeans IDE bằng cách :

	Chọn File -> Open Project (hoặc nhấn tổ hợp phím Ctrl + Shift + O) -> sau đó di chuyển đến thư mục chứa thư mục AIChessProject-> chọn AIChessProject-> Open Project

       - Tiến hành chạy Project trên NetBeans bằng cách vào Run -> Run Project hoặc nhấn nút F6 trên bàn phím.
 
  + Cách 2:

       
- Tạo một Project mới trong NetBeans bằng cách chọn File -> New Project hoặc nhấn tổ hợp phím Ctrl + Shift + N, sau đó copy mã nguồn trong thư mục 'File mã nguồn' (sau khi đã giải nén) và paste vào thư mục Source Packages của Project vừa tạo.
 
      - Tìm tới file MainFrame, click chuột phải vào file, chọn Run File (hoặc click chuột trái vào file MainFrame rồi nhấn tổ hợp phím : Shift + F6).

b) CÁCH 2: Chạy trực tiếp chương trình bằng file .jar
 - Máy tính cần cài đặt JavaSE 1.8 trở lên. Các bước thực hiện bạn tiến hành như CÁCH 1.
 - Sau khi hoàn thành các bước ở trên,có thể chạy trực tiếp chương trình bằng file AIChessProject.jar trong thư mục 'Executable_File'.
